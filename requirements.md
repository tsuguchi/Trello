# Trello風タスク管理ツール 要件定義書

作成日: 2026-04-20

---

## 1. 目的・背景

Trello風のカンバン方式タスク管理ツールを自作する。

**3つの目的を兼ねる:**
1. 自分自身で日常的に使うため
2. オンラインスクールの課題として提出
3. GitHubで公開するポートフォリオ作品

## 2. ターゲットユーザー

- 自分自身（メインユーザー）
- 登録した他のユーザー（各自がアカウントを作って自分のボードを管理できる）
- 将来的にはボード共有機能で複数人共同編集にも対応したい（Phase 3）

## 3. 機能要件

段階的に実装する。

### Phase 1: MVP（まずはここまで）

- [ ] ユーザー登録・ログイン（メール＋パスワード）
- [ ] ログアウト
- [ ] 複数のボード（プロジェクト）を作成・削除できる
- [ ] ボード内にリスト（カラム）を追加・削除できる
- [ ] リスト内にカード（タスク）を追加・編集・削除できる
- [ ] カードをドラッグ&ドロップで別のリストへ移動できる
- [ ] 同じリスト内でカードの順序を並び替えできる
- [ ] **カードにラベル（色タグ）をつけられる**
- [ ] データがクラウドに自動保存される（ブラウザを閉じても残る）

### Phase 2: ポートフォリオ映え

- [ ] カードに期日を設定できる／期限切れの色変化
- [ ] カード検索（タイトルで絞り込み）

### Phase 3: 発展（余裕があれば）

- [ ] ボードを他ユーザーと共有・共同編集
- [ ] チェックリスト
- [ ] コメント
- [ ] ダークモード

## 4. 非機能要件

- **対応ブラウザ**: 最新版の Chrome / Edge / Safari / Firefox
- **レスポンシブ対応**: PC・タブレット・スマホすべてで閲覧と編集ができる
  - スマホ（〜640px）：リストを縦スクロールで表示
  - タブレット（641〜1024px）：リストを横並びで表示
  - PC（1025px〜）：ボード全体を一望できるレイアウト
- **パフォーマンス**: 画面操作がもたつかない程度（厳密な数値目標は設定しない）
- **セキュリティ**: 他ユーザーのデータを見られない（Firebaseのセキュリティルールで制御）
- **アクセシビリティ**: 基本的なキーボード操作に対応（Tabキーでフォーカス移動、Enterで決定など）。セマンティックなHTML要素（`<button>`, `<form>`など）を使う

## 5. 技術スタック

| 役割 | 採用技術 |
|------|----------|
| 言語 | TypeScript |
| フロントエンド | React（Viteでセットアップ） |
| スタイリング | Tailwind CSS |
| 認証 | Firebase Authentication |
| データベース | Firebase Firestore |
| ホスティング | Firebase Hosting または Vercel |
| バージョン管理 | Git / GitHub |

## 6. 画面構成（予定）

1. **ログイン / 新規登録画面**
2. **ボード一覧画面**（ログイン後の最初の画面）
3. **ボード詳細画面**（リストとカードが並ぶメイン画面）
4. **カード詳細モーダル**（タイトル・説明・期日・ラベルなどを編集）

## 7. データ設計（Firestoreイメージ）

```
users/{userId}
  - email
  - displayName
  - createdAt

boards/{boardId}
  - ownerId
  - title
  - createdAt

boards/{boardId}/lists/{listId}
  - title
  - order

boards/{boardId}/lists/{listId}/cards/{cardId}
  - title
  - description
  - dueDate
  - labels
  - order
  - createdAt
```

  - 概要
  - 使い方
  - 技術スタック

  - セットアップ手順

## 9. スコープ外（今回は作らない）

- モバイル専用UI（レスポンシブ対応はするが専用アプリは作らない）
- 通知機能（メール・プッシュ通知）
- 添付ファイル・画像アップロード
- カレンダー連携
- Slack/他サービス連携
- 担当者（assignee）機能（個人利用メインのため）
